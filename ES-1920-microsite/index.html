Project Description

For Once is an audio-visual project that focuses on producing a music video using only the P5JS web editor. A big part of the focus in this project was to integrate the use of a MIDI controller. Therefore, the codes were programmed in a way that I was able to use my MIDI controller to have more control over how the visuals would react to the music over time. The music was composed by myself and I wanted to create a track that helps one to focus, which is also why visually, this project is minimal and repetitive. The intention of keeping it raw and minimal, in my opinion, helps to evoke the calmness and mindfulness that I wanted to achieve. As someone who is new to the visual field, this project has sparked new interest in me, and I look forward to continuing my explorations in aesthetic visuals.

Have a look at my process log, my p5js sketches!

My name is Nysah and I am a percussionist and an aspiring sound designer studying in Lasalle College of the Arts. In all honesty, I have never done or worked with visuals prior to this project. However, thanks to this project, I find myself interested to continue venturing out in this field of work in time to come, on top of music which I am currently active in.
